Exporter Vos Données
Si vous n'avez pas l'intention de mettre à niveau votre compte, vous pouvez exporter vos données collectées en suivant ces étapes:
1. Connectez-vous à votre compte DataWinners dans www.datawinners.com
2. Choisissez un de vos projets DataWinners
3. Cliquez sur l'onglet "Données"
4. Cliquer sur l'onglet “Journal de Soumission"
5. Choisissez "Exporter" dans le coin supérieur droit
6. Répétez cette procédure pour chacun de vos projets de collecte de données
