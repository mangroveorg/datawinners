Requirements for Builder:
===============
The following is only for development environment setup for Builder. To use builder, all that is needed is generated and available out of the box.
To setup development environment, we need to follow few steps.

1. Install npm (Node Package Manager) based on your Operating system
2. run command - "npm install" from base project directory. This will install all dependencies, locally.
3. All javascript files are available under /media/javascript/questionnaire_builder/src directory.
